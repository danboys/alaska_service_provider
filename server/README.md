# Downloading json file with node.js

firebase에서 object 데이터를 불러온 뒤 Node Server를 이용하여 json 파일을 만들고 다운로드 한다.

## Build Setup

``` bash
# move directory 
cd server

# install dependencies
npm install

# starting the Node Server
npm start
```
